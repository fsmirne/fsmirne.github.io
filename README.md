# smirne.com

Personal website for Flavio Smirne, hosted on GitHub Pages.

## Structure

- `index.html` — Home page with bio, quotes, social links, and notable affiliations
- `resume/` — Resume page with embedded PDF viewer
- `css/` — Stylesheets (theming, layout, mobile navigation)
- `js/` — Mobile navigation toggle
- `img/` — Social icons, favicon, and background image

## Setup

The site is served via GitHub Pages with a custom domain (`smirne.com`), configured through the `CNAME` file.
